🗄️ 物业管理费年份表初始化

此工具用于初始化 property_years 表并插入基础年份数据。

操作说明：

  1. 首先点击 "初始化年份表" 按钮创建 property_years 表
  2. 然后点击 "添加2026年份" 按钮插入2026年数据
  3. 完成后即可在费用管理页面使用年份功能

如果以上按钮无效，可以手动执行以下SQL：

-- 创建物业费年份表
CREATE TABLE IF NOT EXISTS property_years (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    year TEXT NOT NULL UNIQUE,
    createdAt TEXT DEFAULT CURRENT_TIMESTAMP
);

-- 插入2026年份
INSERT OR IGNORE INTO property_years (year) VALUES ('2026');
